Bem-vindo à plataforma: Explorando Guapimirim !

Nesta plataforma, você vai explorar as riquezas naturais de Guapimirim,
visitando lugares especiais e encontrando animais escondidos.

Como funciona:

1- Explorando os lugares.
Você vai começar escolhendo um local da lista de pontos turísticos de Guapimirim. Cada lugar é único e tem muitas coisas para explorar.

2- Encontrando os animais.
Dentro de cada local, existem seis animais escondidos. Use o mouse ou a navegação do teclado (TAB) para encontrar esses animais na tela. Se você estiver com accessibilidade ativa, as teclas TAB, ESC e ENTER irão ajudar a navegar melhor entre os animais.

3- Interagindo com os animais.
Ao encontrar um animal, clique nele para abrir um cartão. Neste cartão, você verá :

1º - Como esse animal se chama.

2º - Uma foto desse animal.

3º - Um botão para ouvir o som que esse animal faz.

4º - Uma descrição desse animal.

Ao clicar na foto do animal, ela vai aumentar, ocupando toda a tela para que você a perceba com mais detalhes.

4- Leitor de tela.
Se você estiver usando um leitor de tela, ou seja, o botão de accessibilidade estiver ativo, o jogo irá se integrar ao leitor proporcionando uma experiência mais divertida e completa

5- Atalhos.

1º - Para voltar ou sair de uma seção da tela, use o botão "ESC" como atalho rápido.

2º - Para navegar entre os elementos, use o botão “TAB” como atalho rápido.

3º - Para selecionar um elemento em foco, use o botão "ENTER".

Sobre o Jogo:

Este jogo foi feito para ajudar todas as crianças a explorarem o mundo da natureza de forma divertida e fácil. Aqui você vai poder conhecer diferentes animais de uma região, aprender sobre eles e escutar os sons que eles fazem!

Com sons e descrições simples e muito interessantes, você vai descobrir o que cada animal gosta de fazer, onde eles vivem e outras curiosidades. O jogo foi criado pensando em todo mundo, para que todas as crianças, mesmo aquelas que possuem limitações para enxergar, possam aprender, se conectar, brincar e se divertir muito de maneira especial.

Para os usuários que precisam de uma experiência ainda mais acessível, existe um botão de acessibilidade. Quando esse botão está ativado, ele ajusta o volume dos sons para se encaixar bem com o leitor de tela, proporciona orientações e descrições mais detalhadas, e ainda oferece atalhos fáceis para navegar pelo jogo dentre outras funcionalidades. Para aproveitar essas melhorias, basta ativar no início do jogo o botão de acessibilidade que fica a direita da tela, e tudo será ajustado para que você possa explorar o mundo da natureza com toda a ajuda necessária!

Na parte à direita da tela do jogo, você encontrará alguns botões que ajudam a controlar diferentes funções do jogo. Aqui está uma explicação de cada um deles:

- Som do Jogo.
Esse botão permite ativar ou desativar o som do jogo. Se quiser ajustar o nível do volume, será preciso alterar o volume diretamente no seu computador.

- Modo de Iluminação.
Com esse botão, você pode mudar a aparência do jogo de modo claro para modo escuro.

- Instruções.
Ao clicar nesse botão, você verá informações importantes sobre como jogar e descobrir mais sobre o jogo.

- Modo de Acessibilidade.
Esse botão ativa o modo de acessibilidade, que oferece ajustes especiais para tornar o jogo mais fácil de usar, como sons adequados para leitores de tela e descrições detalhadas dos elementos do jogo.
