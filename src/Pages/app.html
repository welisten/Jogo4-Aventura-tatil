Bem-vindo à plataforma: Explora Guapimirim !

Nesta plataforma, você vai explorar as riquezas naturais de Guapimirim, visitando lugares especiais e encontrando animais escondidos.

Como funciona:

1. Explorando os lugares:

Você vai começar escolhendo um local da lista de pontos turísticos de Guapimirim. Cada lugar é único e tem muitas coisas para explorar.

2. Encontrando os animais:

Dentro de cada local, existem seis animais escondidos. Use o mouse ou a navegação do teclado (TAB) para encontrar esses animais na tela. Se você usar um leitor de tela, as teclas TAB, ESC e ENTER irão ajudar a navegar melhor entre os animais.

3. Interagindo com os animais:

Ao encontrar um animal, clique nele para abrir um card. Neste card, você verá:

  • Como esse animal se chama.
  • Uma foto desse animal.
  • Um botão para ouvir o som que esse animal faz.

Ao clicar na foto do animal, ela vai aumentar, ocupando toda a tela para que você veja com mais detalhes.

4. Leitor de tela:

Se você estiver usando um leitor de tela, o jogo irá se integrar ao leitor proporcionando uma experiência mais divertida

6. Atalhos:

  • Para voltar ou sair de uma seção da tela, use o botão "Esc" como atalho rápido.
  • Para navegar entre os elementos, use o botão “Tab” como atalho rápido.
  • Para selecionar um elemento em foco, use o botão.
